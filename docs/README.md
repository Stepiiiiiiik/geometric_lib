# Math formulas
## Area
- Circle: S = π * R²
- Rectangle: S = a * b
- Square: S = a²
- Triangle: S = a * h / 2

## Perimeter
- Circle: P = 2 * π * R
- Rectangle: P = 2 * ( a + b)
- Square: P = 4 * a
- Triangle: P = a + b + c
  
# Описание проекта
В проекте реализованы геометрические функции для вычисления площади и периметра четырех различных геометрических фигур: квадрата, прямоугольника, треугольника и круга. Эти функции могут быть использованы для выполнения расчетов в различных приложениях, где требуется определить характеристики указанных фигур.

# Код программы на языке python с примерами вывода

## Circe
~~~python
import math


def area(r):
    '''
    Возвращает площадь круга

            Параметры:
                r (int) : радиус окружности
                
                Возвращаемое значение:
                    Площадь окружности, заданной радиусом
    '''
    return math.pi * r * r


def perimeter(r):
    '''
    Возвращает перметр круга

            Параметры:
                r (int) : радиус окружности
                
                Возвращаемое значение:
                    Периметр окружности, заданной радиусом
    '''

    return 2 * math.pi * r

### Пример работы программы
print(area(10))
print(perimeter(10))
314.1592653589793
62.83185307179586
~~~

## Rectangle
~~~python
def area(a, b) :
    '''
    Возвращает площадь прямоугольника

            Параметры:
                a (int) : длина первой стороны прямоугольника
                b (int) : длина второй стороны прямоугольника

            Возвращаемое значение:
                Площадь прямоугольника, заданного введенными сторонами
    '''
    return a * b


def perimeter(a, b) :
    '''
    Возвращает периметр прямоугольника

            Параметры:
                a (int) : длина первой стороны прямоугольника
                b (int) : длина второй стороны прямоугольника

            Возвращаемое значение:
                Периметр прямоугольника, заданного введенными сторонами

    '''
    return 2 * (a + b)


### Пример работы программы
print(area(10, 5))
print(perimeter(10, 7))
50
34
~~~

## Triangle
~~~python
def area(a, h) :
        '''
        Возвращает площадь треугольника

            Параметры:
                a (int) : длина основания треугольника
                b (int) : длина высоты треугольника

            Возвращаемое значение:
                Площадь треугольника, заданного высотой и основанием

        '''
    return a * h / 2


def perimeter(a, b, c) :
        '''
        Возвращает периметр треугольника

            Параметры:
                a (int) : длина первой стороны треугольника
                b (int) : длина второй стороны треугольника
                c (int) : длина третьей стороны треугольника

            Возвращаемое значение:
                Петиметр треугольника, заданного тремя сторонами

        '''

    return a + b + c


### Пример работы программы
print(area(10, 3))
print(perimeter(10, 6, 7))
15
23
~~~
